Toshiba Software IP Speech Codec.

The Software IP Speech Codec was designed for Toshiba TLCS-870 & TLCS-900 Family of CISC microcontrollers.

The speech samples provided are created using a 16kHz sampling rate. Sound quality can be improved upon by using a high quality filter or by means of a D/S converter.

For detailed information about this product please read the "SpeechCodecDevManual.pdf" & "VoiceCompressionSummary.pdf" documents located in the "..\Data sheets\SpeechIP" folder & also consult the User Manual.
